# Boston Housing Statistical Analysis

## Overview

This project analyzes the Boston Housing Prices dataset using Python and Pandas while demonstrating core programming concepts such as exception handling, loops, functions, dictionaries, recursion, and file handling.

## Dataset

- **Dataset:** Boston Housing Prices
- **Rows:** 506
- **Columns:** 14
- Download: https://www.kaggle.com/datasets/vikrishnan/boston-house-prices

Place `Boston.csv` in the project folder before running the program.

## Features

- Safe CSV loading with exception handling
- Data exploration and filtering
- Missing value handling
- Feature engineering
- Interactive price range search
- Neighborhood-wise analysis
- Manual and Pandas statistical calculations
- Weighted average computation
- Automatic text report generation

## Output

The program generates:

- Console-based analysis
- `boston_housing_analysis_report.txt`

## Technologies

- Python
- Pandas
- functools

This project was developed as part of a Python Programming course to demonstrate practical data analysis, statistical computations, and core Python concepts through the analysis of the Boston Housing Prices dataset.
